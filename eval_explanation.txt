Evaluation function for depth limited search:
Evaluation function calculates the utility value when it reaches the specified depth. The eval function tries to find the optimal path using minmax algorithm so as to make player 1(max) win the game. It calculates scores i.e. scores of Player 1 and 2 with respect to player 1.

Score:	It gets points if any possible consective 4 moves of same player on game board either horizontally, vertically or along the two main diagonals.
